# 4. Interpretation & Impact on Design

### **4.1 Bias Categories Identified**
Using combined quantitative + qualitative data, four structural bias frames were extracted:

1. **Functional weakening**  
2. **Plot instrumentalisation**  
3. **Visual gendering / over-emphasis on appearance**  
4. **Loss of narrative agency**

These categories are used to design **three EEG-compatible task structures**:
- **Choice tasks** (responding to biased dialogue)  
- **Evaluation tasks** (judging unfair depictions)  
- **Decision tasks** (acting under biased or incomplete information)

Each task is assigned a **time-locked EEG event window** to enable precise epoch segmentation.
